== Description ==

The Helloprint Store plugin connects to the world's largest platform for customised print products, it's the perfect way to boost your business with the help of Woocommerce. 

It's easy to use, just plug in your account information and you're good to go! You'll be able to search within the Helloprint product offering (over 10million product variations), easily add new products and apply your desired margin. No need to worry about changing prices and managing products with complicated data structures. 

**Do note that this is a commercial plugin that requires a subscription, starting at 79 euro per month.**

And since it's built on top of Woocommerce, you can also use its features like: 

* Product categories 
* Product tags 
* Product descriptions 
* Product images

== Getting Started ==

Adding Helloprint products is easy. Just follow these steps:

1) Make sure your API key is filled in within the plugin details, in case you don't have an API key yet please email api@helloprint.com
2) Create a new product
3) Select type ‘helloprint product’
4) Search for the desired product
5) Set your margin
6) Publish/test the product
7) Set mode to production in plugin settings
8) Start selling
9) Submit an order to Helloprint via Order Actions

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/helloprint` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress admin
3. You will add API key into WP->HelloPrint and save.
